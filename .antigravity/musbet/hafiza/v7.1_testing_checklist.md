## TEST 3: Performans Degerledirme Audit Logging

1. Login as evaluator user
2. Create a NEW performance evaluation (Fill all criteria)
3. Save it
4. Verify:
   - ✓ Record appears in performans_degerledirme table
   - ✓ performans_degisim_loglari has entry with islem_tipi='INSERT'
   - ✓ alan_adi='agirlikli_toplam_puan'
   - ✓ yeni_deger matches saved score

Then:
5. Open same evaluation record again
6. Change score/criteria
7. Save (UPDATE)
8. Verify:
   - ✓ performans_degisim_loglari has NEW entry with islem_tipi='UPDATE'
   - ✓ eski_deger has old score, yeni_deger has new score
   - ✓ surum incremented by 1

Database verification SQL:
```sql
SELECT id, islem_tipi, alan_adi, eski_deger, yeni_deger, degisim_tarihi 
FROM performans_degisim_loglari 
WHERE degerlendirme_id = (SELECT MAX(id) FROM performans_degerledirme)
ORDER BY degisim_tarihi;
```
